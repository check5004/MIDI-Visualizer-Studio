# MIDI Visualizer Studio Repository

MIDIビジュアライザー作成・再生アプリケーション「MIDI Visualizer Studio」のモノレポです。

## ディレクトリ構成

- **[midi_visualizer_studio](./midi_visualizer_studio)**
  - アプリケーションのソースコードが含まれるFlutterプロジェクトです。
  - 開発、ビルド、実行はこのディレクトリ内で行います。
  - 詳細は [midi_visualizer_studio/README.md](./midi_visualizer_studio/README.md) を参照してください。

- **[詳細外部設計書 v2.3 - MIDI Visualizer Studio.md](./詳細外部設計書%20v2.3%20-%20MIDI%20Visualizer%20Studio.md)**
  - アプリケーションの仕様や設計に関するドキュメントです。

## 開発の始め方

Flutterプロジェクトのディレクトリに移動して開発を開始してください。

```bash
cd midi_visualizer_studio
flutter pub get
flutter run
```
